# 딥러닝 기반 원/달러 환율 예측
-글로벌 이벤트와 뉴스 감성의 활용-

## 개요
이벤트 지표와 뉴스 감성 지표를 활용하여 **USD/KRW 종가**를 예측하기 위한 연구
여러 입력 피처 조합(베이스라인, GDELT, 뉴스 요약 감성, 둘 다)과 다양한 모델(LSTM/GRU/CNN_LSTM/CNN_GRU), 시퀀스 길이(lookback), 시프트(shift), 시드(seed)를 조합해 **MSE/MAPE**로 성능 비교
